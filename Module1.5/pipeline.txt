    ML Pipeline মানে কী?

    👉 Step-by-step process automation:

            Data → Preprocessing → Model → Prediction

            👉 সবকিছু একসাথে bundle করা = Pipeline 

            🔹 কেন Pipeline দরকার?

                    ❌ Manual করলে:
                            বারবার code লিখতে হয়
                            ভুল হওয়ার chance বেশি

                    ✅ Pipeline করলে:
                            clean code
                            reusable
                            professional
    🔹 Step by Step Code
                    from sklearn.pipeline import Pipeline
                    from sklearn.preprocessing import StandardScaler
                    from sklearn.linear_model import LinearRegression
                                                    Explain:
                                                    Pipeline → step গুলো একসাথে run করার জন্য
                                                    StandardScaler → data normalize করার জন্য
                                                    LinearRegression → model train করার জন্য

                    # Pipeline তৈরি
                    pipe = Pipeline([
                        ("scaler", StandardScaler()),   # data normalize
                        ("model", LinearRegression())   # model
                    ])

                    # Data
                    X = [[1], [2], [3], [4], [5]]
                    y = [10, 20, 30, 40, 50]

                    # Train
                    pipe.fit(X, y)

                    # Predict
                    pred = pipe.predict([[6]])
                    print(pred)
    🔹 কী হচ্ছে এখানে?
                👉 StandardScaler()

                👉 data scale করে (0 mean, unit variance)
                👉 model ভালো কাজ করে

    👉 Pipeline([...])

             👉 step গুলো এক লাইনে connect করছে

     Visualization Idea

                     [60.]
